***The Single Model Template***

This is a template for writing your own fitting function in Igor. Very little programming knowledge is required. Open the template "SingleModelTemplate_Sphere.pxt", and follow the instructions which show a working model for scattering from a sphere. Instructions are given to show you what to modify for your own calculations.
